Используй агента brainstormer для обсуждения идеи: $ARGUMENTS

Передай агенту полный текст задачи. Агент должен:
1. Исследовать кодовую базу (Glob, Grep, Read) чтобы понять текущее состояние
2. Сгенерировать минимум 3 варианта решения (MVP, сбалансированный, амбициозный)
3. Оценить каждый по бизнес-ценности, сложности и рискам
4. Предложить неочевидные идеи
5. Задать вопросы для уточнения

После получения результата от агента — выведи его ответ пользователю как есть.
Это начало диалога — пользователь может обсуждать, уточнять, просить углубиться в конкретный вариант.
