Проведи аудит UI всего проекта на соответствие дизайн-системе.

Используй агента designer:

1. Прочитай docs/design-system.md и ui-ng/src/styles/_tokens.scss
2. Найди все Angular компоненты со стилями:
   Glob: ui-ng/src/app/**/*.scss
3. Проверь каждый файл на:
   - Использование SCSS-токенов вместо хардкода значений
   - Наличие всех состояний (hover/active/disabled/loading/error)
   - Консистентность отступов (кратность 4px)
   - Доступность (aria-label в HTML, контраст цветов)
   - Responsive поведение (media queries)
4. Проверь shared компоненты в ui-ng/src/app/shared/components/
   на полноту состояний и соответствие дизайн-системе

Выдай отчёт:
## Соответствует дизайн-системе ✅
## Нарушения (файл, строка, проблема, способ исправить)
## Отсутствующие состояния компонентов
## Приоритет исправлений
- Критично (сломан UI или нарушен контраст)
- Важно (несоответствие дизайн-системе)
- Желательно (улучшения UX)
